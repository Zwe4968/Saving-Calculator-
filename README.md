# Savings Money Calculator

A GUI-based savings calculator with animated chart visualization using Python, Tkinter, and Matplotlib.

## Features

- **User-friendly GUI** - Input fields for initial money, monthly savings, and duration
- **Animated Chart** - Watch your savings grow with smooth animations
- **Live Updates** - Real-time display of current month and savings amount
- **Looping Animation** - Animation repeats continuously
- **Savings Summary** - Shows final amount and total saved

## Requirements

- Python 3.7+
- tkinter (usually comes with Python)
- matplotlib
- numpy

## Installation

1. Clone this repository:
```bash
git clone https://github.com/Zwe4968/savings-calculator.git
cd savings-calculator
```

2. Install dependencies:
```bash
pip install matplotlib numpy
```

## Usage

Run the application:
```bash
python Assigmenet_3.py
```

1. Enter your **Initial Money** (starting amount)
2. Enter **Monthly Saving** (amount to save each month)
3. Enter **Number of Months** (duration of savings)
4. Click **Calculate & Show Graph** to see the animated chart
5. Click **Clear** to reset and try different values

## How It Works

- The chart animates line-by-line, showing each month's cumulative savings
- A pulsing red marker highlights the current data point
- An annotation box displays the current month and amount
- The animation loops continuously for ease of viewing
